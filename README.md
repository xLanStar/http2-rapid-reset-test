# Server

A simple http2 server

### How to run

- `cd server`
- `npm i`
- `node app.js`

\*Note: The Node v18.18.2, v20.8.1 and later version already fixed CVE-2023-44487

# Client

A simple http2 client that send /GET /foo and RST

### How to run

- `cd client`
- `pip -r requirement.txt`
- `python main.py`
